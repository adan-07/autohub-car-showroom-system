# 🚗 AutoHub — Car Showroom Management System

A complete multi-role web-based car showroom management platform 
with an integrated AI chatbot, built as a 4th semester project.

## Features
- 4 Role-based Dashboards — Admin, Salesman, Mechanic, Customer
- Role-Aware AI Chatbot — personalized help per user role
- Car Comparison Tool — compare specs, prices & features
- Test Drive & Purchase Booking System
- Service Request Management for mechanics
- Real-time Notifications System
- Dark Mode Support
- Favorites & Saved Cars

## Tech Stack
- PHP
- MySQL (normalized to 3NF)
- Bootstrap
- XAMPP

## Setup
1. Install XAMPP
2. Clone/extract project into `htdocs` folder
3. Import database from `/database` folder into phpMyAdmin
4. Run via `localhost` in browser
